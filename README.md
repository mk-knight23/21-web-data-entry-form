## Validation Features
- **First/Last Name**: Required, minimum 2 characters, letters only
- **Email**: Required, email format validation
- **Employee ID**: Optional, format validation (EMP-XXX)
- **Phone**: Optional, phone number validation (10+ digits)
- **Location**: Optional, minimum 3 characters
- Real-time validation on blur and change
- Visual error states with ARIA attributes
- Clear all functionality
